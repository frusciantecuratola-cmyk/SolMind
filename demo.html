Verifiability Design
The AI agent computes SHA-256(reasoning + allocations + timestamp + price) before submitting the transaction. This hash is stored as last_decision_hash: [u8; 32] in the Solana vault account, along with an IPFS URI to the full reasoning JSON. Anyone can: (1) fetch the hash from chain, (2) download the IPFS reasoning, (3) recompute the hash — proving the AI made exactly that decision with no tampering.
Smart Contract Safety
The Rust program enforces: allocations must sum to 10,000 bps · no single position exceeds (risk_tolerance + 20)% · agent must meet minimum confidence threshold · vault can be paused by authority for human override.
